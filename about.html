About

e-GiantSoft, Inc. 是由一群資訊業專業軟體技術人共同集資成立為兼具理論與實戰經驗的資訊系統軟體及資訊服務之整合公司。 自成立以來以「堅持建構優質的軟體文化、開發卓越企業資訊軟體」為最高理念。 怡智人有個夢，秉持於軟體專案開發領域達到「最佳品質與價值、承諾必實現」之信念，以期成為台灣金融、政府機關、醫療單位等的首選資訊服務品牌。

e-GiantSoft, Inc. 深知人才是軟體資訊開發廠商的根基(正如管理大師威爾許管理名言: "人對了，事情就對了")，所以我們選才為是，怡智對工程人員採取的菁英政策，尋求有能力、有實力的系統開發人員為工作團隊外，更不忘時時保持軟體開發技能的再升級，從鼓勵員工隨時自我學習外定期舉辦教育訓練課程，邀請各領域專家傳授技能及知識。資訊軟體科技是一個時時進步、刻刻演進的行業，唯有保持一顆隨時成長的心才不至於讓資訊浪潮所淹沒。也唯有時時嚴格的為人力品質把關，才能確保提供客戶卓越的資訊系統，也才能符合使用者的需求。

e-GiantSoft, Inc. 所專長之開發技能，跨越不同技術層級與平台：從資訊庫管理系統(Oracle; MS SQL Server; MY SQL..等)、系統開發程式語言(C#、VB DOT NET、Delphi...等)、系統分析CASE Tools(PowerDesigner)；作業平台方面從微軟 Windows 平台、.NET 平台、Google Android 等多樣化的解決方案。應用程式從 Web Solution 、Windows Application、Client/Server、n-Tiers、SilverLight、WPF...都有實際專案開發及上線的經驗。
